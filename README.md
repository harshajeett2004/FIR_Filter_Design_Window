# FIR_Filter_Design_Window
This MATLAB project visualizes and compares the frequency-domain characteristics of four commonly used window functions in Digital Signal Processing (DSP).

## Features

- Rectangular Window
- Hamming Window
- Hanning Window
- Blackman Window
- Frequency Response Analysis
- Magnitude Response Comparison

## Technologies Used

- MATLAB
- Signal Processing Toolbox


## How to Run

1. Open MATLAB.
2. Open the `Matlab Code` folder.
3. Run `Window_DSP.m`.
4. The program will generate the plots for all window functions.

## Output

- Magnitude Response
- Frequency Response
- Comparison of Different Windows
