# FIR-Filter-Design---MATLAB

This MATLAB project analyzes and compares the frequency-domain characteristics of commonly used window functions in Digital Signal Processing (DSP). The program generates both magnitude and phase response plots for Rectangular, Hanning, Hamming, and Blackman windows.

---

## Features

- Frequency spectrum analysis of Rectangular Window
- Frequency spectrum analysis of Hanning Window
- Frequency spectrum analysis of Hamming Window
- Frequency spectrum analysis of Blackman Window
- Phase response visualization for all window functions
- User-defined window length (N)
- Magnitude response displayed in decibels (dB)
- Comparative analysis using MATLAB subplots

## How to Run

1. Open MATLAB.
2. Open the project folder.
3. Run `Window_DSP.m`.
4. Enter the desired window length **N** when prompted.
5. The program will generate:
   - Magnitude Spectrum
   - Phase Response
   - Comparison of all four window functions

---

## Output

The program generates eight plots:

- Rectangular Window Spectrum
- Rectangular Window Phase
- Hanning Window Spectrum
- Hanning Window Phase
- Hamming Window Spectrum
- Hamming Window Phase
- Blackman Window Spectrum
- Blackman Window Phase
